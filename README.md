# Java Swing CRUD Application

CRUD-приложение (напоминающим очень сильно упрощённый
функционал программы Microsoft Excel) для фиксации результатов
производительности схемы .

## Функционал
* Добавление, удаление, очистка записей.
* Валидация ввода.
* Синхронизация с БД.
* Горячие клавиши.

## Запуск
1. Установите JDK 8 или выше.
2. Скомпилируйте проект: `javac *.java`.
3. Запустите: `java Main`.
